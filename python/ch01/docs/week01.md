# [Week 01] Anaconda Spyder (Python IDE) 使用說明

[cite_start]本週重點在於熟悉開發環境。Spyder 是一個專為科學運算與數據分析設計的整合開發環境 (IDE)，掌握其工具能大幅提升您的程式撰寫效率 [cite: 1]。

## 一、 介面導覽
[cite_start]Spyder 的介面主要分為三個核心區域 [cite: 3]：

### 1. 左側：程式碼編輯器 (Editor)
* [cite_start]**功能**：主要撰寫 `.py` 檔案的地方 [cite: 4, 5]。
* [cite_start]**特性**：支援語法高亮、自動補全（按 `Tab` 鍵）與代碼摺疊 [cite: 5]。

### 2. 右下角：IPython 控制台 (Console)
* [cite_start]**功能**：顯示程式執行結果的地方 [cite: 6, 7]。
* [cite_start]**互動**：可直接在此輸入單行指令並按 `Enter` 立即執行（類似計算機功能）[cite: 7]。

### 3. 右上角：多功能面板
* [cite_start]**變數瀏覽器 (Variable Explorer)**：最強大的功能！能即時查看程式中產生的所有變數、資料框 (DataFrame) 或陣列，雙擊即可像 Excel 一樣查看數據內容 [cite: 9, 10]。
* [cite_start]**檔案瀏覽器 (File Explorer)**：管理專案與資料夾檔案 [cite: 11]。
* [cite_start]**說明 (Help)**：查詢函式的用法（將游標停在函式上按 `Ctrl + I`）[cite: 12]。

---

## 二、 常用操作與執行方式
[cite_start]在 Spyder 中，您可以根據需求選擇不同的執行層級 [cite: 14]：

| 功能 | 快捷鍵 (Win) | 說明 |
| :--- | :--- | :--- |
| **全部執行** | `F5` | 執行目前編輯器中的整個檔案。 |
| **執行選取區塊** | `F9` | 只執行滑鼠反白（選取）的那幾行程式。 |
| **Cell 執行** | `Ctrl + Enter` | 執行目前的代碼塊（以 `#%%` 分隔的區塊）。 |
| **中斷執行** | `Ctrl + C` | 程式卡死或跑太久時，在控制台按此鍵強制停止。 |

## 三、 進階必學技巧

### 1. 工作目錄設定 (Working Directory)
[cite_start]在介面右上角設有資料夾路徑。若程式需讀取同資料夾下的檔案（如 CSV），務必點擊資料夾圖示切換至正確路徑，否則會出現 `FileNotFoundError` [cite: 17, 18]。

### 2. 使用代碼塊 (Cells)
[cite_start]在程式碼中輸入 `#%%` 會產生橫線將代碼分塊 [cite: 19, 20]。
* [cite_start]**優點**：可分段測試程式。例如「區塊 A」讀取大檔案，「區塊 B」進行計算。修改計算邏輯時只需重複跑「區塊 B」，不需重新讀取檔案，節省大量時間 [cite: 21, 42]。
